## Highlighted Projects
**JobOrca**: Built a full-stack job tracking platform using Next.js and PostgreSQL to manage and analyze thousands of job applications through dashboards, metrics, and reminder workflows. The platform includes custom APIs built with Next.js and secure authentication using Firebase Google login with session-based access control. To streamline data entry, a custom Firefox and Chrome browser extension was developed to allow one-click saving of job postings directly into the platform. The application was deployed on Vercel with automated CI/CD pipelines powered by GitHub Actions, while the PostgreSQL database was managed using Supabase, and the platform is live at JobOrca.com.

**DiabetiScan**: Developed an end-to-end machine learning web application to predict diabetes using TensorFlow in Python, with a FastAPI backend and a .NET frontend for user interaction. The project included building an ETL pipeline with PySpark to clean real-world health data, handle outliers, and manage missing values to ensure reliable model performance. The application was containerized with Docker and deployed on AWS ECS Fargate, while the trained model was deployed to an Amazon SageMaker endpoint to support real-time inference. Infrastructure provisioning and deployment were fully automated using Terraform to manage AWS resources in a scalable and reproducible manner.
